IPL  MatchWinner Predictor
The IPL ( Indian Premier League ) is India’s premier professional cricket league, usually held between April and May every year. Currently, 8 teams are a part of this competition, and each pair of teams face off twice every season, one each at their respective home grounds. 
This is a tool to predict the winner of an IPL game, given the Stadium Name, Home Team, Away team, Toss Winner and Toss Decision. 
It used the Random Forest Classifier and Kernel Support Vector Machine.
The dataset contains the Stadium Name, Home team, Away team, Toss Winner and Toss Decision data of matches from seasons 2008 - 2019. 
The accuracy of the models on test sets are: 
1. Random Forest classifier = 0.639
2. Kernel Support Vector Machine = 0.621
